hyprotein.simulation.create(config, mode=448, permission=True)¶

Create a current workspace for running simulations.

Simulation tree structure:

├── simulation
│   ├── pdbs
│   ├── mdp
│   └── experiments
│       ├── experiment 1
│       │   ├── analysis
│       │   └── outputs
│       └── experiment 2 (etc.)
│           ├── analysis
│           └── outputs
Parameters

  • config (str) – YAML configuration file, with or without extension declared.

  • mode (octal) – Octal mode file descriptor for permissions, like chmod command. Default: 0o7000 (owner can read, write and execute).

  • permission (bool) – Allows weather or not it will give permission to change the file’s mode settings.
